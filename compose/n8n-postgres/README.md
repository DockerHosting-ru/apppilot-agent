# n8n с PostgreSQL

## Описание
n8n - это мощная платформа автоматизации с открытым исходным кодом, которая позволяет создавать сложные workflows. Этот шаблон разворачивает n8n с базой данных PostgreSQL для надежного хранения данных workflows.

## Включенные сервисы
- **n8n** - Основная платформа автоматизации
- **PostgreSQL 15** - База данных для хранения workflows и настроек

## Переменные окружения

### Обязательные
- `POSTGRES_PASSWORD` - Пароль для базы данных PostgreSQL
- `N8N_BASIC_AUTH_USER` - Имя пользователя для входа в n8n  
- `N8N_BASIC_AUTH_PASSWORD` - Пароль для входа в n8n

### Опциональные
- `N8N_ENCRYPTION_KEY` - Ключ шифрования для production использования

## Системные требования
- **Минимум**: 1GB RAM, 5GB диск
- **Рекомендуется**: 2GB RAM, 10GB диск

## После развертывания
1. Откройте n8n в браузере по адресу `http://your-server:port`
2. Войдите используя указанные логин и пароль
3. Начните создавать ваши первые workflows!
